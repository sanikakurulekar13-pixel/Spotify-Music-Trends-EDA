# Spotify Music Trends EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Spotify music data using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Correlation Analysis
- Genre Analysis
- Popularity Trends
- Visualizations

## Dataset
Spotify dataset from Kaggle.

## Visualizations Included
- Genre Distribution
- Popularity Histogram
- Correlation Heatmap
- Scatter Plots
- Tempo Distribution

## Conclusion
The project identifies important trends and relationships between song attributes and popularity.
